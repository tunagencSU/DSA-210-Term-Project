# 1. Motivation and Objective

The objective of this project is to conduct a comprehensive spatial and temporal analysis of 30 prominent camping and highland locations in the Black Sea Region. For a balanced modeling approach, these locations are evenly distributed across the Eastern (10), Central (10), and Western (10) Black Sea sub-regions. The primary goal is to build a predictive machine learning model that integrates dynamic variables such as transportation infrastructure, meteorological conditions, terrain status, human mobility, and the probability of encountering wildlife. The developed model aims to present the human density and weather conditions for the desired months, along with the overall risk table generated by calculating these variables.
